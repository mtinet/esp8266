### 사용전 세팅방법  
- Arduino IDE 설치  
- 툴 - 보드 - 보드매니터 - esp8266 검색 - 클릭 후 install  
- 보드를 컴퓨터에 USB케이블로 연결  
- 보드를 선택 - NodeMCU 1.0(ESP-12E Module), ESP-12F는 ESP-12E모델의 성능 향상 모델이므로 함께 사용가능  
- 포트를 선택 - 연결된 포트를 확인한 후 선택  
- 아래 사진의 pinout을 확인한 후 스케치를 작성하고 업로드
